==========================================================
  多機能マインドマップアプリ ― 全機能リスト
  （対応プラットフォーム: Android / Windows デスクトップ）
==========================================================

「ただのマインドマップ」では終わらない、学習・情報整理・作業のための
オールインワン・ワークスペースです。アイデアを線でつなぐだけでなく、
動画・PDF・資料・AI・カレンダーまで 1 つのキャンバスに集約できます。

※ 主に Android と Windows デスクトップ向けです（iOS / macOS / Linux /
   Web は一部設定のみ）。コメント・UI は日本語が中心です。


----------------------------------------------------------
■ マインドマップ（コア機能）
----------------------------------------------------------
・フォルダ → ページ（マップ）→ ノードの 3 階層で大量の情報を整理
・ノードを線（コネクション）で自由につなぐ
・接続はドラッグ＆ドロップ時の「アンカー吸着（スナップ）」で自動生成
・ノードの種類: メモ / YouTube / リンク / 添付ファイル（画像・動画等）/
  表（テーブル）
・各ノードに PDF・サイトのメモ（ページ／位置に紐付け）を複数保持
・複数ノードを 1 つにまとめる「格納ノード」（Ctrl+G）、解除も可能
・ノードに別ページを埋め込む「サブマップ」＋ブラウザ風の戻る/進む履歴
  （Alt+←/→）
・図形・装飾の挿入: 直線 / 矢印 / 長方形 / 楕円 / 波線 / 塗りつぶし四角 /
  円 / 中抜き円（テキスト付き可）
・自動レイアウト／整列
・付箋（名前付きグループ）: 色・フォント・サイズ・行数・余白などを設定
・ノードの伸ばせる方向の切替（2 方向 / 4 方向 / 8 方向）
・色分け（順番に色を変える / 色を固定 など）、フォント・サイズ調整
・範囲選択（Shift+ドラッグ）・複数選択（Ctrl+クリック）・一括編集・一括 AI
・「カット（切断）モード」: 2 クリックで線を横切って接続を切る／グループ分割
・スナップ（アンカー吸着・兄弟ノード整列ガイド）
・ドラッグ移動、Undo / Redo（Ctrl+Z / Ctrl+Y、ページ単位スナップショット）
・コピー / 貼り付け、選択ノードを別ページ／新規マップへコピー
・連番タイトルの自動入力（数字・漢数字・ローマ数字・ひらがな・カタカナ）
・基準位置の設定／基準位置へ移動（Ctrl+B / Ctrl+Shift+B）
・クイック子要素生成（個数指定）
・ページの背景画像設定
・ページのお気に入り登録・非表示・並び替え（シャッフル／元に戻す）
・JSON でマップを書き出し／読み込み（画像・PDF を埋め込んでポータブル化）
・フォルダを PC のディレクトリと連携して自動書き出し
・ホーム画面（Android）／デスクトップ（Windows）に、特定のマップを直接開く
  ショートカット（アプリ風アイコン）を作成
・クリップボード画像の貼り付け（貼り付けサイズを % 指定）
・60 種類以上のキーボードショートカット（後述。個別に有効／無効の切替・
  キー割り当て変更が可能）


----------------------------------------------------------
■ 動画・メディア
----------------------------------------------------------
・YouTube 動画の検索・再生・埋め込み（再生リスト／チャンネル対応）
・チャンネルの自動補充キュー、フィルタ（最低再生数・含める/除外キーワード）
・再生リストの動画を、まとめて／1 本ずつノードに埋め込み
・ローカル動画（mp4 / webm 等）の再生（最大 16 倍速まで対応）
・動画のオフライン保存（ダウンロード）と保存済み動画の管理
・ピクチャインピクチャ（PiP）でマップ操作しながら再生（Android / Windows）
・バックグラウンド再生（画面オフでも音声継続：主に Android、ロック画面操作可）
・動画を見ながらのメモ（タイムスタンプ連動で頭出し再生）、メモ履歴保存
・動画タイトルの自動取得（oEmbed）

  ── アプリ内ブラウザ（タブ最大 15・タブに動画／ページのタイトルを表示）──
  ・＋ボタンや Ctrl+U で新しいタブ
  ・Ctrl+クリック（中クリック）でリンクを新しいタブで開く
  ・Ctrl+W でタブを閉じる、Ctrl+Shift+T で閉じたタブを開き直す
  ・右クリック／サイトボタンから Google 検索・Instagram・X(Twitter) も開ける
  ・お気に入り登録したタブは保持。前回開いていたタブは起動時に自動復元
  ・再生速度の変更

■ ドキュメント（閲覧・編集）
----------------------------------------------------------
・PDF ビューア（Flutter ネイティブ描画）
  - ページ単位のメモ（位置マーカー付き）、フォルダ分け、ピン留め
  - ハイライト（行単位・色指定）
  - ページ送り・ページジャンプ・PDF 内テキスト検索
  - 最後に開いたページの記憶、ページの非表示
・Word(docx) / Excel(xlsx) / PowerPoint(pptx) / CSV / テキストの閲覧・編集
・表計算エディタ（数式エンジン内蔵：四則演算・関数・セル参照）
・プレゼン（pptx）のプレゼンターモード
・画像エディタ（注釈・テキスト・図形の追加）
・Markdown / LaTeX（数式）表示
・空ファイルのテンプレート作成（txt / csv / xlsx / docx / pptx）
・docx / pptx → PDF として保存（PDF 書き出し）
  ※ 日本語 PDF を出すには NotoSansJP フォントの配置が必要
・ビューア横に AI／メモのサイドパネル（左右入れ替え可・F6）
  - Web ページではメモは初期非表示、PDF では初期表示（必要時に開閉）
・全メモを一覧表示（PDF / サイト / ノードのメモをタブで切替）
・サイト（URL）ごとのメモ、Google 検索メモ


----------------------------------------------------------
■ AI 機能（生成 AI 連携）
----------------------------------------------------------
・対応プロバイダ: Gemini（中心）/ OpenAI / Anthropic(Claude) / Grok / DeepSeek
・モデル一覧の自動取得・更新、モデルティア（Flash / Pro 等）切替
・キーワードやテーマから AI がマップ（ノード階層）を自動生成
・概念をわかりやすく解説してマップ化（網羅的解説・中断/再開対応）
・資料・ファイルの要約を新規ページに生成
・PDF / ファイルからマインドマップを生成（Max プラン）
・AI クイズ生成（学習用、過去問の重複回避・履歴）
・ページや資料についての追質問（フォローアップ Q&A）
・ノード単位の AI: 解説 / 子ノード自動生成 / カスタム質問（一括 AI シート）
・ノードの AI ボタン（2 モード。右クリック PC／長押し モバイルで切替）
  - 質問モード: API キーを使わず、ノードのタイトルをブラウザの AI
    （ChatGPT / Gemini / Perplexity / Claude / Grok / Google）に渡して開く。
    使う AI を選んで保存（次回も同じ AI）
  - 子要素生成モード: 設定済み API キーで、親ノードの内容に合わせて子を生成
・トークン使用量・コスト（USD / 円）の自動集計、使用量リセット
・生成パラメータ設定（深さ・子/孫の個数・モデルティア）
・生成中のキャンセル（バックグラウンド実行で並行作業可）
・開発者への問い合わせ（バグ報告・要望、1 日の送信上限あり）


----------------------------------------------------------
■ カレンダー / 予定・通知
----------------------------------------------------------
・Google カレンダー風の 1 日タイムライン表示（埋め込み型／全画面型）
・タイムラインの空き枠タップで予定追加、ドラッグで時間移動
・ドラッグで 0 時・24 時をまたいで前日・翌日へもシームレスに移動
  （予定がカーソルに追従）
・予定の追加 / 編集（タップ）／ 削除（選択して Backspace・Delete）
・予定の色分け、祝日表示、タイムゾーン（時差）設定
・繰り返しパターンでの予定の一括コピー（ルーティン）
・予定のシステム通知（リードタイム指定・重複なし・1 件につき 1 回）
・グループで共有できるクラウドカレンダー（メンバーの共有可否切替）
  ※ Google カレンダーとの直接連携は非対応（アプリ内のカレンダー機能です）


----------------------------------------------------------
■ クラウド同期・共有
----------------------------------------------------------
・クラウド同期（ページ・カレンダー）— Firestore REST API + 匿名認証
・8 桁のグループコードで、ページ／予定を仲間と共有
・添付ファイル（画像・動画）のクラウド保存（Firebase Storage）
・複数端末・複数人での連携、ページの自動同期登録
・プランごとのアップロード／ダウンロード／保存容量の上限管理
・無料プランのクラウドデータには保持期限（TTL）あり（有料は延長）
・表示名・アバターの設定
※ アクセス制御はクライアント側実装（本番運用では別途サーバ側ルールが必要）


----------------------------------------------------------
■ 集中ロック（フォーカス支援）
----------------------------------------------------------
・指定時間や曜日スケジュールで操作を制限し、集中をサポート
・全画面オーバーレイ＋ Android の画面固定（lock task / screen pinning）で
  ホーム・履歴ボタンも抑止（端末側の画面固定設定が必要）
・ジオフェンス: 指定地点から半径 N km の出入りでロックを自動起動（GPS）
・解除ボタンの非表示オプション、完了通知

==========================================================
  技術・使用ライブラリ
==========================================================
Flutter（Dart）製。状態管理は単一の Provider に集約し、ローカル保存は
SharedPreferences、クラウドは Firebase の REST API を直接利用しています。

── フレームワーク / 基盤 ──
・flutter / flutter_localizations ... 本体フレームワーク・多言語化基盤
・provider ......................... 状態管理（ChangeNotifier）
・shared_preferences ............... ローカル設定・データの永続化
・uuid ............................. ID 生成
・http ............................. Firebase / 生成 AI / oEmbed 等の REST 通信
・cupertino_icons .................. アイコン

── WebView / ブラウザ ──
・flutter_inappwebview (6.0.0) ..... モバイルの WebView（YouTube・検索・AI 等）
・webview_windows (0.2.2) .......... Windows の WebView（WebView2 ベース）

── PDF / ドキュメント / 画像 ──
・syncfusion_flutter_pdfviewer ..... ネイティブ PDF 表示（ページ追従・ジャンプ）
・pdf .............................. docx / pptx → PDF 書き出し
・csv / excel / archive ............ CSV・xlsx・pptx（zip+xml）の読み書き
・file_picker ...................... 画像 / PDF / 音声などのファイル選択
・open_filex ....................... ファイルを外部アプリで開く
・desktop_drop ..................... デスクトップへのファイルのドラッグ＆ドロップ
・super_clipboard .................. クリップボードの画像/テキスト取得（Ctrl+V）
・flutter_colorpicker .............. 色の選択

── 動画 / 音声 ──
・video_player ..................... ローカル動画再生（最大 16 倍速）
・youtube_explode_dart ............. YouTube 動画のダウンロード
・video_thumbnail .................. 動画サムネイル生成
・audio_service / just_audio ....... バックグラウンド再生（Foreground Service）
・audioplayers ..................... アラーム音の再生
・wakelock_plus .................... 再生中・タイマー中の画面/CPU の維持

── 通知 / 権限 / デバイス ──
・flutter_local_notifications ...... Android の予約通知（zonedSchedule・正確アラーム）
・timezone ......................... 通知時刻の TZ 指定
・local_notifier .................. Windows の OS 通知（トースト）
・permission_handler .............. 権限要求・バッテリー最適化除外
・android_intent_plus ............. メーカー固有の自動起動許可画面へのジャンプ
・geolocator ...................... 位置情報（ジオフェンス集中ロック）
・device_info_plus ................ 端末情報
・window_manager .................. Windows のウィンドウ制御

── 音声入力 / 課金 / 外部連携 ──
・speech_to_text .................. 音声入力（スピーチ to テキスト）
・purchases_flutter (RevenueCat) .. アプリ内課金（Android/iOS/macOS）
・url_launcher .................... 外部 URL / Windows の Web 決済リンク起動
・path_provider ................... 保存先ディレクトリの取得

── 開発時のみ（dev_dependencies）──
・flutter_lints ................... 静的解析ルール
・json_serializable / json_annotation / build_runner
   ............................... JSON 用（※現状はコード自動生成は使わず手書き）

── バックエンド / 外部サービス ──
・Firebase: Firestore（REST API）/ Storage（REST）/ 匿名認証
  （cloud_firestore プラグインは使わず http で直接アクセス）
・生成 AI: Gemini / OpenAI / Anthropic(Claude) / Grok / DeepSeek の各 REST API
・課金: RevenueCat（モバイル）/ Stripe Web 決済リンク + REST（Windows）

── ネイティブ実装 ──
・Android: Kotlin（MainActivity）で画面固定（lock task）と
  ホーム画面ショートカット（ShortcutManagerCompat）を実装
・Windows: WebView2 / local_notifier / syncfusion PDF / window_manager

── キー（env.json でビルド時注入。アプリには直接埋め込まない）──
・Firebase 各種キー、Gemini API キー、RevenueCat 公開キー 等
  （`--dart-define-from-file=env.json` で渡す。未設定でもアプリは起動し、
   該当機能のみ無効化される）
